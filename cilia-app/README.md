<h1> Cilia Challenge </h1>

## Tecnologias utilizadas:

```
- Bootstrap
- Highcharts
- Leaflet
- Sass
- React
```

## Como visualizar o projeto:

Instale os pacotes

```
npm install
```

Execute o projeto:

```
npm start
```

No browser acessar a aplicação:

```
http://localhost:3000
```
